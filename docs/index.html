How It Works

Trigger

Press the global shortcut from anywhere. Morph reads your clipboard automatically.

2

Rewrite

AI rewrites your text using your custom system prompt. Results stream in instantly via Groq.

3

Copy

One click to copy the rewritten text back to your clipboard. Done.

⌘ + ⇧ + M

Works from any app. Morph appears, rewrites, and disappears. Zero friction.

Install on macOS

The app is unsigned. Follow these steps after downloading:

1
Open the .dmg file and drag Morph into Applications.
2
Remove the quarantine flag by running this in Terminal:
xattr -cr "/Applications/Morph.app"
3
Open the app normally. You're all set!

Install on Windows

Windows SmartScreen may block the unsigned installer:

1
Run the downloaded .exe installer.
2
If SmartScreen appears, click "More info".
3
Click "Run anyway" to proceed.
